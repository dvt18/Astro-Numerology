## 🚀 Local Development Setup

To boot this project locally:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Launch Dev Server**:
   ```bash
   npm run dev
   ```
   Open **`http://localhost:5173`** in your browser to view the application live.

3. **Verify Production Build**:
   ```bash
   npm run build
   ```
